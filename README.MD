# jTro - Geo Django

### Framework and Libraries used
-

**Frontend**
* Html, CSS, JS
* Bootstrap, Leafletjs, jquery

**Backend**
* GeoDjango
* PostGIS (middleware)
* GeoServer

**Server**
* Nginx
* Gunicorn


### Features
-
* Web Mapping
* Zoom in/Zoom out
* Location Search
* Layer control
* Print map
* View in full screen
* Zoom to layer
* Map scale
* Map coordinate
* Marker cluster


**Start Geo Server (Linux Debian)** ``sh /usr/share/geoserver/bin/startup.sh``
